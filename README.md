# Docker flask app

Is a little project (CRUD) using flask and docker

## Installation

[Insall](https://www.docker.com/) docker and docker-compose

## Usage

1. Build image

```bash
 docker build -t flaskapp:1.0 .
```
2. Run image
```bash
 docker-compose build
 docker-compose up 
```
